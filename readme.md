## Remote Keyboard

Install and start the server:

~~~
npm install
npm run build
npm start
~~~

## Development

In one tab, watch and recompile the react frontend:

~~~
npm run watch
~~~

Keep the server open in another tab. Server updates will require to restart this
process:

~~~
npm start
~~~
